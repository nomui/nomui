## 何时使用

- tooltip 工具提示，可以通过 styles 属性配置外观

## API

### Tooltip

| 参数             | 说明                 | 类型      | 默认值  |
| ---------------- | -------------------- | --------- | ------- |
| align            | 层对齐位置           | `string`  | `top`   |
| ignoreMouseEvent | 忽略鼠标移入移出事件 | `boolean` | `false` |
