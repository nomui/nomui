区隔内容的分割线。

## 何时使用

- 对不同章节的文本段落进行分割。
- 对行内文字/链接进行分割，例如表格的操作列。

## API

| 参数        | 说明                       | 类型                      | 默认值       |
| ----------- | -------------------------- | ------------------------- | ------------ |
| value      | 默认值                   | `string`                 | -      |
| data | 图标数据源           | `array`     |
| allowClear       | 是否允许显示清空按钮 | `boolean`                 | `true`      |
| searchable       | 是否允许搜索 | `boolean`                 | `false`      |
| iconRender        | 自定义图标展示函数，为true时不再以默认字段风格显示           | `()=>{}`  | - |
