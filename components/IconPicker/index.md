图标选择器

## 何时使用

- 需要选择图标类型时

## API

| 参数        | 说明                       | 类型                      | 默认值       |
| ----------- | -------------------------- | ------------------------- | ------------ |
| value      | 默认值                   | `string`                 | -      |
| data | 图标数据源           | `array`     |
| allowClear       | 是否允许显示清空按钮 | `boolean`                 | `true`      |
| searchable       | 是否允许搜索 | `boolean`                 | `false`      |
| iconRender        | 自定义control图标渲染           | `()=>{}`  | - |
| itemRender        | 自定义下拉框选项渲染函数          | `()=>{}`  | - |
| popupWidth        | 下拉框宽度          | `number`  | 340 |
