漫游式引导

## 何时使用

- 用于需要创建一系列提示来引导用户操作界面时

## API

| 参数          | 说明             | 类型                          | 默认值 |
| ------------- | ---------------- | ----------------------------- | ------ |
| steps          | 步骤数组         | `array`                      | -      |
| padding          | 高亮区域周边的阴影边距           | `number`                      | 2      |



## step item

| 参数          | 说明             | 类型                          | 默认值 |
| ------------- | ---------------- | ----------------------------- | ------ |
| target          | 步骤指向的dom元素或者nomui组件实例         | `dom element \| ()=>{}`                      | -      |
| title          | 步骤标题         | `string`                      | -      |
| description          | 步骤描述           | `string`                      | 2      |
| render          | 自定义步骤内容渲染           | `function`                      | -      |
