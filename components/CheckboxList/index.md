复选框组合，用于包裹一组 CheckBox。

## API

| 参数    | 说明       | 类型                                   | 默认值 |
| ------- | ---------- | -------------------------------------- | ------ |
| options | 单选项配置 | `[{text:string,value:number\|string}]` | -      |

## Method

| 参数     | 说明             | 类型                     |
| -------- | ---------------- | ------------------------ |
| setValue | 设置复选框的数据 | `array：[value1,value2]` |
| getValue | 取得复选框的数据 | `array：[value1,value2]` |
