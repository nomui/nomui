下拉框组件

## 何时使用

- 需要选择颜色的时候。

## API

| 参数          | 说明                                    | 类型              | 默认值  |
| ------------- | --------------------------------------- | ----------------- | ------- |
| data | 自定义数据源，默认格式为[{id:'#aaa'}]                          | `array` | - |
| useStyles     | 是否使用原子类作为数据源，同时取值也会是原子类名，配置为false时才可使用自定义数据源                      | `boolean`          | `true`  |
| popupWidth         | 弹出层宽度                          | `number`         | `192` |
| onClick       | 分割按钮模式下，左边按钮的点击回调      | `()=>{}`          | -       |
