下拉框组件

## 何时使用

- 需要选择颜色的时候。

## API

| 参数          | 说明                                    | 类型              | 默认值  |
| ------------- | --------------------------------------- | ----------------- | ------- |
| data | 自定义数据源，默认格式为[{id:'#aaa'}]，当useHex为true时有效                          | `array` | - |
| useHex     | 是否使用hex色值作为数据源，同时取值也会是hex色值，默认为false并使用原子类名称作为数据源与值                      | `boolean`          | `false`  |
| popupWidth         | 弹出层宽度                          | `number`         | `192` |
